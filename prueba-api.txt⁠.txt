La API fue construida utilizando Node.js y Express. Se configuró el middleware express.json() para permitir que el servidor interprete objetos JSON enviados en el cuerpo de las peticiones. Se definieron dos endpoints (rutas) de tipo POST: '/registro' e '/incidencia', los cuales capturan los datos enviados por el cliente a través del objeto 'req.body' y responden confirmando la recepción de la información mediante un objeto JSON.
